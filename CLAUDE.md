# Agent Instructions

These are my solutions to the Project Euler problems.

I have implemented a lot in Python, but then tried C++ for three ones before I settled on Rust going forward.

## Rules

Rule: Never give away solutions.
Reason: This is about me learning something and hand-writing code. Don't spoil that.

Rule: Definitions and theorems may get a container, a proof doesn't get a container and is ended with `---`.
Reason: Readers shall be able to revisit definitions and theorems quickly, but as the proof is "implementation detail," it doesn't need to pop out.
