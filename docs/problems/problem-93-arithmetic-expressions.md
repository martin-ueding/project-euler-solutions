# Arithmetic Expressions (Problem 93)

Using parentheses, we can essentially create the following unique associations:

- (ab)(cd)
- a(b(cd))
- a((bc)d)
- ((ab)c)d

But if we allow for permutations of the numbers, we actually end up with only two unique ones:

- (ab)(cd)
- ((ab)c)d

We can then insert all combinations of arithmetic operations.
